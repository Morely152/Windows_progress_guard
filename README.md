# Windows进程守护

## 软件简介

**Windows进程守护软件是一款用于监控和管理Windows系统中关键进程的工具。该软件可以自动启动指定的进程，确保它们在运行过程中被意外终止时重启这些进程。该软件具备以下功能：**
- **进程守护**：当监控的进程意外终止时，自动重启进程。
- **开机自启动**：设置软件和被监控的进程开机时自动启动。
- **自动最小化到托盘**：启动后自动最小化，安静守护进程。
